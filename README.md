# A/B Test Analysis for Subscription Conversion

## Overview

This project analyzes A/B test results for a mobile app subscription screen.

The goal was to compare conversion rates between control and test groups, evaluate statistical significance, and estimate business impact.

## Business Case

Current subscription conversion rate: 17%.

A new screen design was tested with a "50% discount" message while keeping the same price.

## Tasks

* Compare conversion rates between groups
* Calculate p-value
* Build 95% confidence intervals
* Interpret business impact
* Prepare recommendation

## Tools

Python, Pandas, NumPy, Statsmodels, Matplotlib, Excel

## Result

The experiment was analyzed using statistical methods and recommendations were prepared based on significance level and conversion uplift.
